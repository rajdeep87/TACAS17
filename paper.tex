\begin{abstract}
%===============================================================================
%
  \emph{Abstract Conflict-Driven Learning} (ACDL) is a combination of
  abstract interpretation and the well-known Conflict-Driven Clause
  Learning (CDCL) algorithm that is used in SAT solvers. ACDL has been
  successfully used with non-relational abstract domains such as
  intervals to improve the performance of a solver for floating point
  arithmetic, for instance. 
%
Whereas the
algorithms typically used in SAT solvers can be rather easily
lifted from the Boolean lattice to other non-relational domains,
lifting them to relational domains is challenging.
%because of inter-"literal" dependencies
%because of the weaker properties of "literals"
%
In this paper we present the adaptations of algorithms and heuristics
for propagation, decision making and conflict learning necessary in
order to enable the use with relational domains.
%
We have implemented these algorithms inside the 2LS verification tool
for the example of the template polyhedra domain.
%
We evaluate the performance of our analyzer in comparison with program verifiers that 
uses propositional-level solvers.  Experimental evidence on a set of 
\rmcmt{N} benchmarks \rmcmt{drawn from X} shows significant reduction 
in the number of decisions and backjumping iterations along with stronger 
deductions and learned clause aided by the richer abstract domains.
%
\end{abstract}
